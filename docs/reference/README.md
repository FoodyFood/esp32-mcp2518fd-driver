# Reference Documents

The PDFs are not committed to this repository. Download them from Microchip and place them here.

## Required files

| Filename | Document | Download |
|---|---|---|
| `External-CAN-FD-Controller-with-SPI-Interface-DS20006027B.pdf` | MCP2518FD Datasheet (DS20006027B) | https://www.microchip.com/en-us/product/MCP2518FD |
| `MCP25XXFD-CAN-FD-Controller-Module-Family-Reference-Manual-DS20005678E.pdf` | MCP25XXFD Family Reference Manual (DS20005678E) | https://www.microchip.com/en-us/product/MCP2518FD |

## Search tool

Once the PDFs are in place, use `docs/search.py` to query both documents:

```bash
python docs/search.py <keyword1> <keyword2> ...
```

Results are written to `docs/reference/search_results.txt`.
